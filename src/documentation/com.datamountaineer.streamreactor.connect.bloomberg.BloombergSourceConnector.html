Kafka Connect Bloomberg is a Source connector to subscribe to Bloomberg feeds via the Bloomberg labs open API and write to Kafka Requires: An active Bloomberg subscription
Step 1: Decide on ticker subscriptions
The format is TICKER:FIELD,FIELD,..; e.g.AAPL US Equity:LAST_PRICE;IBM US Equity:BID 
    connect.bloomberg.subscriptions=AAPL US Equity:LAST_PRICE,BID,ASK;IBM US Equity:BID,ASK,HIGH,LOW,OPEN
Step 2: Set the correct authentication
The mode to authentication against the Bloomberg server. Either APPLICATION_ONLY or USER_AND_APPLICATION. 
    connect.bloomberg.authentication.mode=APPLICATION_ONLY
Step 3: Connect to the correct end-point
After setting the host and port, select the Bloomberg service to connect to. Can be //blp/mkdata or //blp/refdata. 
    connect.bloomberg.service.uri=//blp/mkdata
